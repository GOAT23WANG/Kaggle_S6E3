# Baseline Consolidation Prompt

你的目标是把 Kaggle_S6E3 的历史最佳单模型路线收敛到统一入口。

执行要求：

1. 读取 src/train.py、src/features.py 与 Archived 中的 V16b 脚本
2. 识别必须迁移的特征组、编码流程、参数配置与输出结构
3. 在不扩大实验范围的前提下，把强基线迁移到统一入口
4. 运行并验证 OOF AUC 是否接近历史强基线
5. 若低于历史 best 超过 0.0005，停止并报告差异来源

禁止事项：

- 不要开始新特征实验
- 不要切换到高成本模型
- 不要在迁移过程中同时重构多个无关模块

产出要求：

1. 明确新的 baseline_id
2. 写入实验日志
3. 输出与历史 V16b 的差异分析
